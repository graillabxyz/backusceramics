# Backus Ceramics

Market Validation Platform for Backus Ceramics.

## Tech Stack
- Next.js
- Tailwind CSS
- Lucide React
- Framer Motion

## Development
This project is set up for automatic deployment to Vercel via GitHub.

## Environment Variables

- `XENDIT_KEY` - server-side Xendit secret key used to create hosted payment sessions for prepaid bookings.
- `XENDIT_PUBLIC_KEY` - public Xendit key reserved for future client-side or embedded payment flows. Do not use the secret key in browser code.
